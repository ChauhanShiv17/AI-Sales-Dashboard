AI Sales Dashboard

An interactive Machine Learning powered sales analytics dashboard built using Python and Streamlit.

Features:
Sales Trend Analysis
Customer Segmentation using K-Means
Sales Forecasting
Interactive Visualizations
Data Cleaning & EDA

Tech Stack:
Python
Streamlit
Pandas
Scikit-learn
Statsmodels
Matplotlib

Project Structure:
AI_Sales_Dashboard/
│
├── data/
├── notebooks/
├── models/
├── app.py
├── analysis.py
├── requirements.txt
└── README.md

Objective:
To help businesses analyze sales trends, segment customers, and forecast future revenue using Machine Learning techniques.
